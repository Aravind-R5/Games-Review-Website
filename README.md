# GameVault — The Ultimate Gaming Review Platform

A premium, full-stack **Gaming Review Platform** built with Django and React, designed for gaming enthusiasts to explore everything from Triple-A blockbusters to Hidden Indie Gems.

⚡ **Tech Stack**

| Layer | Technology |
| :--- | :--- |
| **Backend** | Python, Django, Django REST Framework |
| **Auth** | Token-Based Authentication (DRF) |
| **Database** | PostgreSQL (Production) / SQLite (Local) |
| **Frontend** | React (Vite) |
| **Styling** | Custom Vanilla CSS (Design System) |
| **Hosting** | Render (Backend) & Vercel (Frontend) |

🎯 **Features**

🔸 **Explore All Genres —** Discover a massive catalog of games across all categories, including major studio releases and independent titles, complete with deep-dive reviews and ratings.

🔸 **Slideshow Experience —** Modern homepage featuring a premium auto-sliding showcase of the latest trending and top-rated games with smooth transitions.

🔸 **Interface & UX —** State-of-the-art gaming design featuring deep gradients, vibrant yellow accents (IMDb-inspired), glassmorphism effects, and responsive mobile-first layouts.

🔸 **Admin Control —** Comprehensive backend management to update the game library, curate featured content, and manage review data.
